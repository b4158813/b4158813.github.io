笔记-燃气轮机原理及其应用

第一单元

燃气轮机

功能

  • 以空气和燃气为工作介质，将热能转变为机械能的高速回转式动力机械

  • 与内燃机、蒸汽轮机一样，为原动机

组成

一般压气机、燃烧室、涡轮（透平膨胀机）、控制系统及基本辅助设备组成

  • 压气机（Compressor）：完成气体压缩过程
  • 燃烧室（Burner）：完成燃烧加热过程
  • 涡轮（透平膨胀机）（Turbine）：完成高温高压燃气的膨胀作功过程

负荷：燃气轮机输出的机械功率用来驱动的发电机、泵、鼓风机螺旋桨或车轮等

工作原理

压气机从外界吸入空气（吸气），压缩到一定压力，送入燃烧室中，与喷入的燃料混合，进行燃烧，生成的高温高压燃气进入透平中膨胀作功，推动涡轮，带动压气机一起旋转，并向外界输出机械功（带负荷），做功后的废气排入大气中或加以利用（排气）

燃料中的化学能->热能->机械功

  • 燃气轮机在涡轮中所做的机械功，大约2/3用来带动压气机；其余1/3机械功则通过机组的输出轴去带动各种负荷

理想燃气轮机循环——布雷顿循环

1->2 等熵压缩，2->3等压吸热，3->4等熵压缩，4->1等压放热

设备结构

压气机（轴流式）

  • 由转子+气缸构成，17~18级叶片，气缸分为上气缸+下气缸

  • 可分为进气缸、压气缸、排气缸

    • 进气缸：与进气过滤装置连接（大气端）
    • 排气缸：与燃烧室连接（透平端）

  • 空气过滤装置需要定期清洗

  • 入口导叶调整进气量

燃烧室

  • 燃烧室和透平承担着1k℃以上的高温
  • 机组容量不同，燃烧室数量不同
  • 基本包括火焰筒、联焰管过过渡段、燃料喷嘴

透平

  • 由转子+气缸组成，一般3~5级，分为上气缸+下气缸
  • 气缸内部圆周上安装静止叶片（喷嘴），静叶组和动叶组构成一级

循环方式

开式循环

  • 以空气为工质，用内燃的方式在燃烧室中加热，并把废气放回大气来排热（开式）

  • 使用广泛

    内燃：燃气作为工质做功（对燃料要求高）

闭式循环

  • 工质加热后，用外燃方式（空气锅炉、原子能反应堆或其他换热器）膨胀作功后，用热交换器排热（闭式）

    外燃：工质通过换热器获得燃料燃烧的热量后做功（燃烧产物不是循环工质，对燃料要求低）

  • 使用并不广泛，工质可采用非空气工质，如 氦气

蒸汽轮机可以看成闭式燃气轮机的一种变相形式

工业过程式循环

  • 工质同其他工业设备有联系的燃机循环

特点

优点

  1. 重量轻、体积小、投资省
  2. 启动快、自动化程度高、操作方便
  3. 水、电、润滑油消耗少、少用或不用水
  4. 燃料适应性强、公害少
  5. 维修快、运行可靠

缺点

  1. 热效率低
  2. 使用的经济性和可靠性较差
